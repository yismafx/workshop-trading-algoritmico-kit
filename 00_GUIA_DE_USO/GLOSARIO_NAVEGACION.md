## 🔗 CONVENCIONES DE NOMENCLATURA

Para entender cómo están nombrados los archivos en este repositorio:

### Patrón de Nombres
```
[Tipo]_[Descripción]_[Categoría].md
```

**Ejemplos:**
- `GUIA_INICIO.md` → GUIA = Tipo, INICIO = Descripción
- `Setup_Colab_Rapido.md` → Setup = Tipo, Colab_Rapido = Descripción
- `Programa_Detallado_Workshop.md` → Programa = Tipo, Detallado_Workshop = Descripción

### Prefijos Comunes
- `GUIA_` → Documento principal de orientación
- `Setup_` → Instrucciones de configuración
- `Programa_` → Estructura del workshop
- `Mapa_` → Recurso visual/diagrama

### Sufijos Comunes
- `_Completa` → Versión exhaustiva
- `_Rapido` → Versión rápida/simplificada
- `_Detallado` → Con información extendida

## ❓ FAQ DEL GLOSARIO

### ¿Por qué cambiaron los nombres de archivo en v2.1?
**R:** Para estandarizar y hacer los nombres más descriptivos. Los nombres antiguos usaban convenciones mixtas (mayúsculas/minúsculas) que causaban confusión.

### ¿Los links antiguos funcionan?
**R:** No. Todos los links fueron actualizados en v2.1. Si encuentras un link roto, por favor repórtalo como Issue en GitHub.

### ¿Cómo encuentro un archivo específico?
**R:** Usa este glosario o el [SITEMAP.md](SITEMAP.md). También puedes buscar en GitHub con `Ctrl+K` o `Cmd+K`.

### ¿Qué significa el emoji 🟡 en "Estado"?
**R:** 
- ✅ Verde = Disponible ahora
- 🟡 Amarillo = En desarrollo (próxima versión)
- ❌ Rojo = Descontinuado (ya no se usa)

### ¿Dónde está la Prompts Library?
**R:** La Prompts Library está en desarrollo activo. Será parte del contenido premium disponible para participantes del workshop.
