## 🎯 RUTAS DE NAVEGACIÓN RECOMENDADAS

### 🟦 Ruta 1: Nuevo Participante (Primera Vez)

**Objetivo:** Entender el workshop y configurar el ambiente

```
1. README.md
   ↓
2. GUIA_INICIO.md (Sección "¿Por Dónde Empezar?")
   ↓
3. Programa_Detallado_Workshop.md (Ver estructura completa)
   ↓
4. Guia_Setup_Completa.md (Decidir qué setup usar)
   ↓
5. Setup_A_Colab_Rapido.md (Recomendado para empezar)
   ↓
6. GUIA_INICIO.md (Volver para siguiente paso)
```

**Tiempo estimado:** 60-90 minutos

---

### 🟨 Ruta 2: Setup Rápido (Quiero Empezar Ya)

**Objetivo:** Estar listo para la primera sesión en <50 minutos

```
1. Setup_A_Colab_Rapido.md
   ↓
2. Ejecutar notebook: Setup_y_Practica_Trading.ipynb
   ↓
3. Validar instalación
   ↓
4. ✅ Listo para S1
```

**Tiempo estimado:** 30-50 minutos

---

### 🟩 Ruta 3: Consulta Durante el Workshop

**Objetivo:** Resolver dudas específicas mientras avanzas

```
GUIA_INICIO.md
    ├─→ FAQ (Sección "Preguntas Frecuentes")
    ├─→ Troubleshooting (Errores comunes)
    └─→ Recursos Adicionales
```

**Tiempo estimado:** 5-15 minutos por consulta
