## 🎯 ¿Qué es Setup B: Python Local?

**Setup B** será la guía completa para instalar y configurar **Python localmente** en tu computadora (sin usar Google Colab), ideal para:

- Traders que prefieren trabajar offline
- Usuarios con limitaciones de conectividad
- Desarrollo avanzado con IDEs como VS Code o PyCharm
- Mayor control sobre el ambiente de desarrollo
